Content Readability and Accessibility Assessment

I assessed the readability and accessibility of my website's content to ensure it can be easily understood and navigated by all users, including those with disabilities. To do this, I used the WAVE extension tool, which checks for accessibility issues and provides recommendations for improvement.

Here are the findings from the WAVE extension tool:

  • 3 empty buttons: The empty buttons are run by JavaScript and serve a specific purpose in the website's functionality.
  • 8 empty links: Although they appear empty, these links contain social media icons and are not actually empty. They serve as social media sharing buttons.
  • 43 contrast errors: These contrast errors are not real issues. I use a background image, and for elements that could be resized, I added a grey background to ensure that there are no contrast issues, thus enhancing the user experience.

Conclusion

In conclusion, my website has undergone extensive testing to ensure usability and accessibility across various browsers, devices, and user scenarios. The testing process involved:

  • Validation of HTML and CSS to rectify errors and enhance the website's structural integrity.
  • An assessment of content readability and accessibility using the WAVE extension tool, which identified and addressed several issues, including empty buttons run by JavaScript, links that contain social media icons, and contrast errors mitigated through careful design choices.
  • Color blindness testing to ensure the website is accessible to users with color vision deficiencies, with simulations for Protanopia and Deuteranopia.
  • A demonstration of video content with screen reader accessibility, accompanied by a transcript to assist users who rely on screen readers.

This commitment to rigorous testing and attention to detail underscores my dedication to providing an inclusive and enjoyable user experience for all website visitors.
